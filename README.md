# 📚 CISA Question Bank (Moodle GIFT Format)

This repository contains a complete question bank for CISA (Certified Information Systems Auditor) based on ISACA Body of Knowledge (BOK).

## 📌 Structure

- Domain 1: Information System Auditing Process
- Domain 2: Governance and Management of IT
- Domain 3: Information Systems Acquisition, Development and Implementation
- Domain 4: Information Systems Operations and Business Resilience
- Domain 5: Protection of Information Assets

## 📊 Features

- 100 questions per domain
- Multiple choice (4 options)
- Answer key included
- Difficulty level: Easy, Medium, Hard
- Explanation for each question
- Format: Moodle GIFT (ready to import)

## ⚙️ How to Use

1. Download `.txt` file from each domain
2. Import into Moodle:
   - Question Bank → Import
   - Format: GIFT
3. Use for quiz / exam preparation

## 🎯 Purpose

This project is created for academic purposes and CISA exam preparation.
